## assignment rules

- prefer matching task tech stack to person's skills
- respect availability -- don't assign to unavailable people
- spread work across the team -- avoid overloading one person
- when assigning to someone other than yourself, always create a handoff doc (see `.foundry/casts/handoff.md`)

## concurrent task limit

max tasks any single person should have in-progress at once: **2**

this limit applies to the execute stage -- the agent will not start new tasks for a person who already has this many in-progress.
